# 02) System Overview

## Platform Scope
Restaurant OS includes:
- POS & payment orchestration
- Orders and kitchen lifecycle
- Inventory and recipe-level consumption
- Menu engineering and pricing control
- CRM and loyalty
- Campaign engine
- Analytics and AI-ready data outputs
- White-label tenant admin

## Operating Model
- API-first cloud platform with tenant-aware services.
- Real-time state propagation for kitchen and order boards.
- Offline-first POS edge client with sync reconciliation.

## Stakeholders
- Tenant owner, operations manager, cashier, kitchen staff, marketer, analyst, customer, platform operator.

## Global Design Drivers
- Strict tenant isolation
- Resilience under peak meal windows
- Regional tax/compliance adaptability
- Low-latency POS interactions
- Auditable transaction history
